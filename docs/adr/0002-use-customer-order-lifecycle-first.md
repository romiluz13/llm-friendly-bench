# Use Customer Order Lifecycle first

The first Interactive Proof Lab scenario will center on Customer Order Lifecycle, with CRM as a first-class variant, rather than starting narrowly with quote-to-cash, ecommerce, healthcare, fintech, or support alone. This keeps the demo instantly familiar to the widest range of customers while preserving enough real enterprise complexity to expose Hidden SQL Cost through accounts, contacts, order intent, line items, fulfillment, payment, exceptions, support interactions, and audit history.
